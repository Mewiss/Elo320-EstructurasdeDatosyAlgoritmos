Funciones en C

  • Encapsula código.
  • Permite reutilizar código.
  • Se pueden compilar por separado.
  • Retorna siempre un tipo de dato específico.

Funciones en C

  • Se puede separar interfaz (prototipo) de implementación.
  • Interfaz (prototipo): Sólo define existencia y formato de función. Suficiente para ser invocada.

//tipo_retorno funcion(parametros con o sin nombre);
float potencia(float a, float b);
//....

calculo = x * potencia(y, 2.5);

Funciones en C

  • Implementación: Define la implementación del código de la función.

/*tipo_retorno funcion(parametros con nombre) {
*   Implementacion...
}*/

float potencia(float a, float b) {
    return pow(a,b);
}

Funciones en C

  • Si se utiliza solamente la implementación, esta tiene que estar definida antes de la función que la utiliza.
  • Si se utiliza el prototipo, este debe estar definido antes de la función que la usa.
  • Prototipo + Implementación: libertad de ubicación de la implementación en diferentes archivos.

Ámbito (Scope)

Corresponde al alcance de las variables, es decir, el segmento en el cual existen y pueden ser usadas → entre { y }.


#include<stdio.h>

void saludo(){ // var i sirve sólo dentro de función saludo.
    int i = 5;
    printf("Hola alumno numero %d", i);
}

int main(){// var a sirve sólo dentro de función main. 
    int a = 10;
    saludo();
    printf("%d\n", a);
    return 0;
}

Variable Local

Definida dentro de un ámbito determinado, entre llaves { y }.


#include<stdio.h>
int main(){// var a sirve sólo dentro de función main. 

    int a = 10;
    if(a==10){
        int b = 1;
        printf("%d %d\n", a, b);
    }

    printf("%d %d\n", a, b); // ¿Qué pasa con la variable b?

    return 0;}

Aplica para todo elemento que contenga llaves.

Variable Global

Está definida fuera de cualquier ámbito. 


#include<stdio.h>

int a=1; //global
int b=2; //global

int main(){
    int a = 10; //local
    b++;
    printf("%d %d\n", a, b);//¿Qué imprime? 
    return 0;
}

Puede ser llamada, modificada y utilizada desde cualquier ámbito, siempre y cuando no exista una variable local con el mismo nombre.

Macro

Valor o conjunto de instrucciones constantes que se definen a nivel de preprocesamiento. No es una variable y no puede ser modificada.


#include<stdio.h>
#define MACRO 10

int main(){
    int a = 5;
    printf("%d\n", a*MACRO);
}

Constante

Una variable constante es aquella que se declara con el keyword const, e implica que ésta no puede ser modificada. Tiene todo el resto de las propiedades de una variable común.


#include<stdio.h>

int main(){
    const int a = 5;
    a = 10; //¿Cómo actúa el compilador?
    printf("%d\n", a);
}

Arreglos en C

Arreglos

  • Conjunto homogéneo y ordenado de un tipo de datos.
  • Se identifican por su posición relativa mediante un índice encerrado con []
  • Parten en el índice 0.

int arr[6]; //6 enteros creados en arr.
arr[3] = 222; //Al cuarto elemento de arr se le asigna un 222.

Recorrer un arreglo

Podemos utilizar cualquiera de las estructuras de control iterativas para recorrerlos.


#include

int main(){
    int i;
    float a[5];
    
    for(i=0;i<5;i++){
        float val; //¿Cuántas veces nace y muere val?
        scanf("%f", &val);
        a[i]=val;
    } //¿Cómo reducir este código?

}

Strings

  • Arreglo de char y el carácter '\0'.

// Los strings terminan con '\0' (fin de string)

str[3] = 'i'; // ¿Qué hace?

Funciones con Strings

  • strlen(s): Retorna el largo de un string s.
  • strcpy(destino, origen): Copia el string origen al string destino.
  • strcmp(s1, s2): Compara los strings s1 y s2. Retorna 0 si son iguales.
