¿Qué es una Estructura de Datos?

Una estructura de datos es una abstracción para organizar datos de manera que puedan ser manejados, procesados y utilizados eficientemente a través de un programa.

Permiten que los algoritmos trabajen de forma eficaz sobre la información.

Ejemplos

  • Palabras → Strings
  • Python: Listas, Tuplas, Diccionarios
  • C++: Vectores, Listas, Map
  • Control+Z → Stack/Pilas
  • Filas en bancos/supermercados → Queues/Colas
  • Gramática de lenguajes → Trees/Árboles

Actividad 1.1.1: Somos personas, tenemos datos distintos

  • En grupos de a 4 personas, deben presentarse y seleccionar 4 datos que los representen como estudiantes de este curso.
  • Por ejemplo: nombre, carrera, hobbies, algo que les gustaría aprender en este curso.

Actividad 1.1.2: Somos personas, tenemos datos distintos

  • Cada grupo comparte los datos seleccionados
  • Discutimos cómo estos datos podrían ser organizados en una estructura de datos
  • Cada grupo agrega la información a la planilla

¿Qué es un Algoritmo?

Secuencia de pasos para resolver un problema o realizar una tarea.

Debe ser preciso, finito y eficiente.

Actividad 1.2.1: Tu algoritmo me lleva

  • Cada grupo elige una tarea cotidiana (ej. caminar a la pizarra, dibujar algo, doblar papel).
  • Escriban un algoritmo paso a paso (tipo pseudocódigo).
  • Intercambien algoritmos con otro grupo.
  • El grupo receptor debe ejecutar literalmente las instrucciones como si fueran una máquina.
  • Luego reflexionamos:
    ¿Qué salió mal? ¿Faltaron instrucciones? ¿Se entendió bien?

Tiempo total: 30-40 min
