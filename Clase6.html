Taxonomía de la memoria

Segmentos de memoria

Resumen

  • Stack → Variables locales, funciones (rápido, limitado).
  • Heap → Memoria dinámica (flexible, debes administrarla).
  • Data → Globales y estáticas (persisten toda la ejecución).
  • Text → Código ejecutable.

Memoria dinámica

Controlada programador y almacenada en el heap de la memoria.

Permite:

  • Modificar tamaño de arreglo
  • Crear estructuras de datos flexibles (listas, árboles, grafos)
  • Administrar memoria en tiempo de ejecución

Memoria dinámica: programación en C

  • Se accede mediante punteros.
  • 3 instrucciones básicas de stdlib.h:
    • malloc(): reserva de memoria.
    • realloc(): reasignación de memoria.
    • free(): liberación de memoria.
  • valgrind: software para debugging de memoria.

Memoria dinámica: malloc

  • Reserva memoria en el heap.
  • Devuelve la dirección (puntero) de inicio.
  • Usa sizeof como base del tamaño.

#include<stdio.h>
#include<stdlib.h>

int main(int argc, char **argv){
    int *arreglo=NULL;
    
    // sizeof(tipo_de_dato) o sizeof(expresion)
    arreglo = malloc(10 * sizeof(*arreglo));
    // arreglo = malloc(10 * sizeof(int));
    // arreglo = malloc(10 * sizeof *arreglo);     
    // arreglo = (int *) malloc(10 * sizeof(int));  //cast opcional en C
    return 0;
}

Memoria dinámica: calloc

  • calloc() reserva memoria y la inicializa en cero.
  • Permite que todos los valores empiecen en cero.

#include <stdio.h>
#include <stdlib.h>
int main() {
    int *arreglo;
    // calloc reserva 5 ints e inicializa todos en 0
    arreglo = (int *) calloc(5, sizeof(int));

    for(int i = 0; i < 5; i++)
        printf("%d ", arreglo[i]); // imprime: 0 0 0 0 0
    
    free(arreglo);
    return 0;
}

Memoria dinámica: realloc

  • Reasigna memoria reservada.
  • Si se expande, puede cambiar de dirección.

#include<stdio.h>
#include<stdlib.h>

int main(int argc, char **argv){
    int *arreglo=NULL;

    arreglo = malloc(10 * sizeof(*arreglo));

    arreglo = realloc(arreglo, 5 * sizeof(int));
    arreglo = realloc(arreglo, 20 * sizeof *arreglo);

    return 0;
}

Memoria dinámica: free

  • Libera memoria reservada en heap.
  • Evita fugas de memoria (memory leaks).

#include<stdio.h>
#include<stdlib.h>

int main(int argc, char **argv){
    int *arreglo=NULL;

    arreglo = malloc(10 * sizeof(*arreglo));
    arreglo = realloc(arreglo, 5 * sizeof(int));
    arreglo = realloc(arreglo, 20 * sizeof *arreglo);

    free(arreglo);
    return 0;
}

Problemas con punteros

Memory Leak

  • Pérdida de acceso a memoria reservada.
  • No se puede liberar hasta fin del programa.
  • Puede saturar la memoria.

int main(int argc, char **argv){
    int *p;
    
    p = malloc(sizeof(int));
    p = malloc(sizeof(int)); // fuga, se pierde la dirección anterior

    free(p);

    return 0;
}

Dangling

  • Puntero a memoria liberada o reasignada.
  • Genera errores impredecibles (segfault, corrupción de memoria).

int main(int argc, char **argv){
    int x=2;
    int *p, *q;

    p = malloc(sizeof(int));
    q = p;

    free(p);  /* q aun mantiene referencia al heap */
    *q = x;   // error

    return 0;
}

Struct en C

  • Un struct permite agrupar variables bajo un mismo nombre.
  • Se accede a los miembros usando el operador . (para variables) o -> (para punteros).

#include <stdio.h>

struct Estudiante {
    char nombre[50];
    char apellido[50];
    float nota;};

int main() {
    struct Estudiante e1;
    strcpy(e1.nombre, "Maria");
    strcpy(e1.apellido, "Gonzalez");
    e1.nota = 7.0;
    printf("%s %s - %.2f\n", e1.nombre, e1.apellido, e1.nota);
    return 0;
}

typedef struct

Con typedef creamos un alias y evitamos escribir struct al declarar variables. 


#include <stdio.h>
#include <string.h>

typedef struct {
    char nombre[50];
    char apellido[50];
    float nota;
} Estudiante;

int main() {
    Estudiante e1; // no necesitamos escribir 'struct'
    strcpy(e1.nombre, "Pedro");
    strcpy(e1.apellido, "Perez");
    e1.nota = 8.5;

    printf("%s %s - %.2f\n", e1.nombre, e1.apellido, e1.nota);
    return 0;
}

Struct y punteros


Estudiante *ptr = malloc(sizeof(Estudiante));
strcpy(ptr->nombre, "Ana");
ptr->nota = 9.0;

printf("%s - %.2f\n", ptr->nombre, ptr->nota);
free(ptr);

Con typedef, el uso de punteros es más legible y no necesitamos escribir struct.

Paso por valor vs paso por referencia con structs


typedef struct {
    char nombre[50];
    char apellido[50];
    float nota;
} Estudiante;

void modificar_por_valor(Estudiante e) {
    e.nota = 7.0; // solo copia
}

void modificar_por_referencia(Estudiante *e) {
    e->nota = 7.0; // modifica original
}
