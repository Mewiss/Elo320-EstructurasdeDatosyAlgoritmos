Diferencia entre %d y %i

En C, %d y %i son especificadores de formato utilizados en funciones como printf y scanf.

  • %d se utiliza para imprimir o leer un número entero en formato decimal.
  • %i también se utiliza para imprimir o leer un número entero, pero puede aceptar números en diferentes bases (decimal, octal y hexadecimal) dependiendo del prefijo del número.

Arreglos

  • Arreglo: Conjunto homogéneo y ordenado de un tipo de datos.
  • Se identifican por su posición relativa mediante un índice encerrado con paréntesis cuadrado ([], parte de índice 0).

int arr[6]; //6 enteros creados en arr.
arr[3] = 222; //Al cuarto elemento de arr se le asigna un 222.

Strings

  • String: Estructura de datos para representar palabras.
  • Arreglo de char y el caracter '\0'.

// Los string terminan con '\0' (fin de string) 
// '\0' ocupa 1 char en el string.
char str[6] = "hola"; //String de largo máximo 5. 
str[3] = 'i'; // ¿Qué hace? 
printf("%s\n", str);
  • Veremos más sobre strings las próximas clases.

Arreglos Multidimensionales e Inicialización

Además, se pueden definir arreglos multidimensionales.

También, los arreglos pueden ser inicializados utilizando { }.


#include

int main(){
    char nom[10] = "Camilo";
    char a[5][20] = {"Juan", "Francisco", "Pedro"};	
    int b[3][3] = {{1,2,3}, {4,5,6}, {7,8,9}}, i, j;

    for(i=0;i<3;i++){
        for(j=0;j<3;j++) //¿Qué pasó con las llaves? 
            printf("%d ", b[i][j]);
        printf("\n");
    }
}
