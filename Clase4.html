Struct en C

  • Un struct permite agrupar variables bajo un mismo nombre.
  • Se accede a los miembros usando el operador . (para variables) o -> (para punteros).

#include <stdio.h>

struct Estudiante {
    char nombre[50];
    char apellido[50];
    float nota;};

int main() {
    struct Estudiante e1;
    strcpy(e1.nombre, "Maria");
    strcpy(e1.apellido, "Gonzalez");
    e1.nota = 7.0;
    printf("%s %s - %.2f\n", e1.nombre, e1.apellido, e1.nota);
    return 0;
}

typedef struct

Con typedef creamos un alias y evitamos escribir struct al declarar variables. 


#include <stdio.h>
#include <string.h>

typedef struct {
    char nombre[50];
    char apellido[50];
    float nota;
} Estudiante;

int main() {
    Estudiante e1; // no necesitamos escribir 'struct'
    strcpy(e1.nombre, "Pedro");
    strcpy(e1.apellido, "Perez");
    e1.nota = 8.5;

    printf("%s %s - %.2f\n", e1.nombre, e1.apellido, e1.nota);
    return 0;
}

Struct y punteros


Estudiante *ptr = malloc(sizeof(Estudiante));
strcpy(ptr->nombre, "Ana");
ptr->nota = 9.0;

printf("%s - %.2f\n", ptr->nombre, ptr->nota);
free(ptr);

Con typedef, el uso de punteros es más legible y no necesitamos escribir struct.

Paso por valor vs paso por referencia con structs


typedef struct {
    char nombre[50];
    char apellido[50];
    float nota;
} Estudiante;

void modificar_por_valor(Estudiante e) {
    e.nota = 7.0; // solo copia
}

void modificar_por_referencia(Estudiante *e) {
    e->nota = 7.0; // modifica original
}

Argumentos por línea de comandos

CLI Arguments: argv y argc

argc y argv: parámetros de ejecución

  • En C, podemos entregar argumentos a los ejecutables mediante la línea de comando.
  • Esta función main contiene dos argumentos: argc y argv.

#include<stdio.h>

int main(int argc, char **argv){ //o *argv[]
    return 0;
}
  • argc: Argument Count → cantidad de parámetros.
  • argv: Argument Vector → los parámetros (strings).

argc y argv: parámetros de ejecución

#include<stdio.h>

int main(int argc, char **argv){ //o *argv[]

    if(argc!=3){
        printf("Debe ingresar 3 parámetros, intentelo nuevamente");
        return -1; // exit(-1) si terminar proceso.
    }

    printf("Argumento 0: %s\n", argv[0]);
    printf("Argumento 1: %s\n", argv[1]);
    printf("Argumento 2: %s\n", argv[2]);

    return 0;
}

argc y argv: transformando tipos

  • atoi() transforma string a int
  • atol() transforma string a long
  • atoll() transforma string a long long
  • atof() transforma string a float.
  • sscanf() lee un string y lo usa como scanf().
  • strtol(): string a long con base (10, 16, 2, ...)
  • strtod(): string a double
  • ⚠️ deben verificar el string!

argc y argv: transformando tipos


#include<stdio.h>
#include<stdlib.h>

int main(int argc, char **argv){ //o *argv[]

    if(argc!=4){
        printf("Debe ingresar 4 parámetros, intentelo nuevamente");
        return -1;
    }

    int a;
    sscanf(argv[2], "%d", &a);

    printf("%d %d %ld\n", atoi(argv[1]), a, strtol(argv[3], NULL, 10));

    return 0;
}

Streams

Streams: Archivos

  • C usa punteros FILE para abrir streams a archivos.
  • fopen() abre el stream; fclose() lo cierra.
  • Modos principales: "r" (lectura), "w" (escritura, sobrescribe), "a" (anexar); agregar + habilita lectura y escritura.

FILE *arch;
arch = fopen("archivo.txt", "r"); // apertura stream
// ... operaciones ...
fclose(arch);                      // cierre obligatorio

Material de apoyo asíncrono: ManejoArchivos.html

Archivos: Escritura y Lectura

  • Escritura:
  • fprintf(): con formato
  • fputs(): string
  • fputc(): caracter
  • Lectura:
  • fscanf(): con formato
  • fgets(): línea (hasta \n)
  • fgetc(): caracter

#include<stdio.h>

int main(int argc, char **argv){
    FILE *arch = fopen("test.txt", "r");
    char linea[256];

    while(fgets(linea, 256, arch) != NULL){
        fprintf(stderr, "%s", linea);
    }

    fclose(arch);
    return 0;
}

Streams: Manejo del puntero

A diferencia de Python, en C se puede manejar el puntero del archivo:

  • rewind(): inicio
  • fseek(): mueve n bytes
  • ftell(): posición actual
  • fgetpos(): guarda posición en fpos_t
  • fsetpos(): mueve a posición guardada

Otros Streams

Existen variados streams en C:

  • Standard I/O: scanf(), printf()
  • Strings: sscanf(), sprintf()
  • Archivos: fscanf(), fprintf()

Tipos de Datos Abstractos TDA

Marie González-Inostroza

Tipo de Datos

  • Colección de valores y las operaciones que se pueden realizar con éstos.
    Ejemplo: enteros (int) con operadores aritméticos.

TDA: Tipos de Datos Abstractos

Modelo que define un conjunto de datos y las operaciones posibles, sin importar la implementación.

  • Interfaz: operaciones disponibles.
  • Implementación interna oculta.
  • Acceso solo mediante la interfaz.

Ejemplos de TDA

  • Pila (Stack): Inserción y eliminación por un único extremo (LIFO).
  • Cola (Queue): Inserción por un extremo y eliminación por el otro (FIFO).
  • Lista: Conjunto ordenado de elementos, con acceso a posiciones específicas.

Ejemplo: Lista como TDA

Interfaz de operaciones de una lista:

  • crear_lista()
  • insertar(lista, pos, valor)
  • eliminar(lista, pos)
  • buscar(lista, valor)
  • imprimir(lista)

La implementación puede variar (arreglos, listas enlazadas, etc.).

Motivación del TDA

Permiten:

  • Diseñar programas más modulares y reutilizables.
  • Separar el qué hace un tipo de dato del cómo lo hace.
  • Facilitar la abstracción y el trabajo en equipo.

Problemas con Arreglos

  • Eficientes y útiles, pero poco flexibles al cambiar tamaño.
  • Eliminar o insertar requiere mover elementos → O(n).
  • Listas enlazadas ofrecen una alternativa más eficiente.

Lista Simple Enlazada

Es un TDA que enlaza nodos a través de punteros.

  • No deben estar ordenados en memoria.
  • No es necesario desplazar elementos: solo cambiar punteros.

Lista Simple Enlazada: Nodos

  • La lista se compone por nodos
  • Los nodos se componen por dato y puntero

struct nodo {
    int dato; // dato
    struct nodo *sgte; // puntero a un nodo
};

typedef struct le_nodo {
    float datos[20];
    struct le_nodo *sgte;
} nodo_t;

void free_list(nodo_t *cabeza){
