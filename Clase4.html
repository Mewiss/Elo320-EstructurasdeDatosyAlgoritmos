Diferencia entre %d y %i

En C, %d y %i son especificadores de formato utilizados en funciones como printf y scanf.

  • %d se utiliza para imprimir o leer un número entero en formato decimal.
  • %i también se utiliza para imprimir o leer un número entero, pero puede aceptar números en diferentes bases (decimal, octal y hexadecimal) dependiendo del prefijo del número.

Diferencia entre %d y %i


#include   
int main() {
    int num1 = 10, num2 = 010, num3 = 0x10; // 010 es octal, 0x10 es hexadecimal
    printf("Decimal: %d\n", num1); // Imprime 10
    printf("Octal: %i\n", num2); // Imprime 8
    printf("Hexadecimal: %i\n", num3); // Imprime 16
    return 0;
}

  • Se identifican por su posición relativa mediante un índice encerrado con []
  • Parten en el índice 0.

Funciones con Strings

  • strlen(s): Retorna el largo de un string s.
  • strcpy(destino, origen): Copia el string origen al string destino.
  • strcmp(s1, s2): Compara los strings s1 y s2. Retorna 0 si son iguales.
